# Huffman-Encoder-Decoder
# Huffman Encoder & Decoder

> 基于哈夫曼树的无损编解码系统 —— 数据结构课程实验  
> 模拟双工信道上 Alice 与 Bob 的安全文本传输

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📖 实验背景

**问题来源：** Alice 和 Bob 需要通过一条只允许传输 0/1 符号的无差错信道进行文本通信。  
**解决方案：** 基于字符频率构建哈夫曼树，生成变长前缀码，实现高效无损编码。双方只要共享同一棵哈夫曼树（通过安全信道交换 `hfmtree.txt`），即可完成编解码，相当于一种**带混淆性质的轻量保密通信**。

本项目完整实现了实验要求的 **初始化、编码、解码、打印代码文件、凹入法打印树、菜单界面** 六大功能。

## 🚀 功能一览

| 功能 | 对应操作 | 输入文件 | 输出文件 |
|------|----------|----------|----------|
| **I** 初始化 | 从字符集文件建立哈夫曼树并保存 | `CharSet.txt` | `hfmtree.txt` |
| **E** 编码   | 利用已建好的树对正文编码     | `ToBeTran.txt` (明码)  | `CodeFile.txt` (0/1串) |
| **D** 解码   | 利用树将编码文件还原为文本   | `CodeFile.txt`          | `TextFile.txt`          |
| **P** 打印   | 在终端显示编码文件（每行50字符），同时写入 | `CodeFile.txt` | `CodePrint.txt` |
| **T** 打印树 | 在终端以凹入法显示哈夫曼树，同时写入 | 内存中的树 | `TreePrint.txt` |
| **Q** 退出   | 结束程序 | —                       | —                       |

所有功能通过循环菜单交互式调用，直至输入 `Q` 退出。

## 📁 文件说明

实验过程中涉及的文件均在项目根目录下运行（可自行配置路径）：

- `CharSet.txt`：字符集大小 n，以及 n 个字符和对应权值（建议包含参考示例数据）
- `hfmtree.txt`：保存的哈夫曼树结构（用于下次启动时直接加载，无需重复初始化）
- `ToBeTran.txt`：待编码的英文文本（大写字母与空格）
- `CodeFile.txt`：编码得到的 0/1 序列
- `TextFile.txt`：解码后复原的文本，应与 `ToBeTran.txt` 完全一致
- `CodePrint.txt`：紧凑格式的 0/1 序列，每行 50 个字符
- `TreePrint.txt`：凹入法表示的哈夫曼树文本
